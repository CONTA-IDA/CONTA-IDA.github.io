분석하고, 만들고, 다시 구현합니다.

관심사

리버스 엔지니어링으로 프로그램의 동작 방식을 따라가고, 반복되는 분석을 줄이기 위한 스크립트와 작은 도구를 직접 만듭니다.

IDA Pro, Ghidra, x64dbg를 활용한 분석과 Python 자동화, 클라이언트 기반 재구현 실험을 중심으로 작업합니다.

Reverse

IDA Pro, Ghidra, x64dbg, binary analysis notes

Develop

Blender, 대체 에셋 설계, 프로토타입 모델링, 레거시 게임 재구현

Workflow

추적한 내용을 재구현으로 옮기는 흐름

  1. Trace

    실행 흐름, 문자열, 상태 변화를 관찰하고 기록합니다.

  2. Automate

    반복 분석은 스크립트와 작은 CLI 도구로 자동화합니다.

  3. Rebuild

    확인한 규칙과 조작감을 작은 프로토타입으로 검증합니다.

  4. Model

    원본 리소스에 기대지 않고 필요한 대체 에셋과 표현 방식을 설계합니다.
