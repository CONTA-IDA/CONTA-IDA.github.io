About

분석하고, 만들고, 시각화합니다.

관심사

리버스 엔지니어링을 통해 프로그램이 어떻게 동작하는지 살펴보고, 분석 과정에서 필요한 스크립트와 작은 도구를 직접 만드는 일에 관심이 많습니다.

IDA Pro, Ghidra 같은 분석 도구와 Python 자동화, 시각화 프로토타입, 레거시 게임 재구현 실험을 함께 다룹니다.

Stack

분석과 제작을 오가는 구조를 선호합니다.

Reverse

IDA Pro, Ghidra, x64dbg, binary analysis notes

Develop

Python tooling, C/C++, JavaScript, Git, documentation

Model

Blender, 대체 에셋 제작, 프로토타입 모델링, 레거시 게임 재구현

Workflow

분석에서 재구현까지 이어지는 흐름

  1. Trace

    동작을 관찰하고 함수 흐름, 문자열, 상태 변화를 기록합니다.

  2. Automate

    반복 분석은 스크립트와 작은 CLI 도구로 줄입니다.

  3. Rebuild

    핵심 규칙과 조작감을 직접 구현해 프로토타입으로 검증합니다.

  4. Model

    원본 리소스에 의존하지 않고 클라이언트 기반 프로토타입에 필요한 대체 에셋과 표현을 설계합니다.
